# udesk-link-demo

使用 url 方式，访问 udesk 客服页面，并写上使用demo

## install
```sh
  # npm
  npm install udesk-js-link

  # yarn
  yarn add udesk-js-link
```

## submodule
```sh
  git submodule update
```
